## 准备工作：

1. 安装`Python`（3.6 或更高版本）

2. [建立虚拟运行环境](https://www.baidu.com/s?wd=virtualenv)（可选）

3. 下载代码

4. 安装依赖：`pip install -r requirements.txt`

5. 根据自己需要修改：`citys.txt`

6. 准备完成，现在可以愉快的去抓取数据啦~

## 运行

运行主文件：`main.py`
